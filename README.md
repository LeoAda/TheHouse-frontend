# The House - Frontend
This repo contains the Frontend code for a personnal project called "The House". It is related to a backend project available [here](https://github.com/LeoAda/TheHouse-backend). It is build using basic html/css/js and Caddy framework. It provides simple forms pages to manage the "database".
This choice of no framework is it keep the project as lightweight as possible and requires the minimum of dependencies to run.

## Run
Require the backend to be running first.
### Local
To run the frontend locally, follow these steps:
Open the `index.html` file in your web browser :).
### Docker
To run the backend using Docker, use the following command:
```bash
docker compose up --build
```
